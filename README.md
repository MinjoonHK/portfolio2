# Portfolio2

여태까지 해온 작업물을 보여주는 포트폴리오 페이지 입니다!

## 💻 프로젝트 소개

기존의 Vanilla JS 로 작성한 포트폴리오 페이지를 React로 리팩토링 하였습니다.<br/>
리팩토링을 진행하며 반응형 디자인과 모달을 추가하여 사용자 경험을 개선하기 위하여 노력하였습니다.

## ⏲️ 개발 기간

2024.05 - 2024.05

### ⚙️ 개발 환경

<ul>
  <li>React.js@18.2.0, TypeScript</li>
  <li>Build: vite</li>
  <li>IDE : VScode</li>
  <li>Deploy: Vercel</li>
</ul>
